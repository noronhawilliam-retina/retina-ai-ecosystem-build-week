# Retina AI Ecosystem

**7 AI products · 2 specialist networks · 5 foundations, tools and ventures · one operating layer**

Retina AI Ecosystem is a working navigator and public architecture map for a coordinated portfolio of AI-enabled business services. It connects seven focused AI products, the Retina and Blockchain specialist networks, Retina Comunicacao, RetinaCoin, Consultoria Blockchain, a Codex-built token generator and the independent AstroConsulta vertical.

This repository is the clean, judge-safe Build Week layer. It contains no production credentials, customer data or private infrastructure code.

## Live demo and video

- Public demo: https://retina-ai-ecosystem.williamnoronha.chatgpt.site
- Demo video: https://youtu.be/eVV9RqHujjY

## Why it exists

Small businesses are often asked to operate disconnected tools and vendors. Retina's individual products already solved focused jobs. During OpenAI Build Week, we created the connective layer that explains and demonstrates how those products work as one approval-first operating system.

## Working features

- Responsive product and architecture map
- Interactive Ecosystem Navigator
- Goal-based specialist and product routing
- Direct maps to both complete specialist networks
- Maturity-aware starting rules
- Delivery cadence recommendations
- Clear separation between existing products and Build Week work
- Links to live public ecosystem surfaces

## Run locally

Requirements:

- Node.js 22.13+
- npm or pnpm

```bash
npm install
npm run dev
```

Open the local URL printed in the terminal.

Production build:

```bash
npm run build
```

No API key, sample data or login is required to test the navigator.

## How to test

1. Open the home page.
2. Scroll to **Ecosystem Navigator**.
3. Switch among the four business goals.
4. Change maturity and preferred pace.
5. Verify that the specialist, product sequence, starting rule and delivery cadence update immediately.
6. Follow any product link to inspect a live public Retina surface.

## Architecture

The demo uses a small Next-compatible React application deployed as Cloudflare Worker-compatible ESM through Vinext.

```text
Business goal + maturity + pace
              ↓
      Ecosystem Navigator
              ↓
 Specialist routing + operating rules
              ↓
Sites · Ads · SEO/GEO · Social · Content · Identity · Agents
              +
Retina Comunicacao · RetinaCoin · Consultoria Blockchain · AstroConsulta
Blockchain Consultants · Codex Token Generator
```

## Codex and GPT-5.6 collaboration

GPT-5.6 in Codex was the primary agentic development environment for this Build Week extension.

Codex helped us:

- audit a multi-product ecosystem and distinguish confirmed capabilities from roadmap items;
- identify the strongest hackathon framing and select the Work & Productivity track;
- design the information architecture and interactive routing model;
- implement the responsive experience and original project imagery;
- apply compliance-safe language to RetinaCoin and avoid speculative token claims;
- verify the submission requirements, deadlines and evidence expectations;
- prepare build, testing, README and Devpost materials as one coherent package.

The human founder made the core product decisions: present the ecosystem as a unified operating layer, retain human approval gates, include the existing products as evidence, and communicate the Brazil eligibility issue transparently.
